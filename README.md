# FogCard Protocol 🎴

> **FogCard (迷雾卡牌)** is a decentralized strategic card game featuring true "Fog of War" on the blockchain, powered by Celestia's Private Blockspace and Verifiable Encryption (ZK Proofs).
>
> 迷雾卡牌是一款基于 Celestia 私密区块空间和可验证加密技术（零知识证明），在区块链上实现真正“战争迷雾”的去中心化策略卡牌游戏。

## 🚀 Core Features

- **True Fog of War**: The opponent cannot see your hand or deployment. (真正的战争迷雾：对手无法看到你的手牌和部署)
- **Verifiable Fairness**: Every hidden move is mathematically proven to follow rules using Zero-Knowledge. (可验证的公平性：每一步隐秘操作都由密码学保证绝对合规)
- **No Servers Needed**: 100% decentralized, running persistently on public networks. (无需服务器：全过程去中心化运行)
- **Anti-Ragequit Liveness**: Smart contracts ensure players can force-settle against disconnected opponents. (防弃赛机制：断线或恶意退出将受到智能合约的强制结算)
